Para executar o projeto, basta compilar as classes Java e executar o 
projeto, executando os seguintes passos:

$ ./compile.sh
$ ./TurboFire <opcao> <opcao> <opcao>

PARA EXECUTAR A VERSAO MESTRE:
$ ./TurboFire master

PARA EXECUTAR A VERSAO ESCRAVA:
$ ./TurboFire slave <ip_maquina_mestre> 3425

3425 e a porta pre definida para o servidor mestre.
